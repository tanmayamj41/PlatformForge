# PlatformForge

PlatformForge is a production-inspired Internal Developer Platform (IDP) built as a learning and portfolio project.

## Goals

- Learn Platform Engineering from first principles
- Build reusable platform components
- Automate infrastructure provisioning
- Improve developer experience
- Demonstrate production-grade engineering practices

## Roadmap

- Linux
- Networking
- Git
- Docker
- Kubernetes
- Terraform
- CI/CD
- Observability
- AI Integration
